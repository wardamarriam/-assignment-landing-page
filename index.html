How to Customize Assignment Links

To modify the links for each assignment card, follow these steps:

  1. Find the assignment card you want to modify in the HTML code
  2. Locate the <a> tag inside the card body
  3. Change the href attribute to your desired URL
  4. Change the text between <a> and </a> to customize the button text

Example:

<a href="YOUR_URL_HERE" class="card-link" target="_blank">YOUR_TEXT_HERE</a>
